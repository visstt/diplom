# Инструкция по запуску

## Backend

1. Перейдите в папку server:

```bash
cd server
```

2. Убедитесь, что установлены зависимости:

```bash
npm install
```

3. Создайте файл `.env` с переменными окружения (если еще не создан):

```
DATABASE_URL="postgresql://user:password@localhost:5432/Titan"
JWT_SECRET=your-secret-key-change-in-production
```

4. Примените миграции базы данных:

```bash
npx prisma migrate dev
```

5. Запустите сервер:

```bash
npm run start:dev
```

Сервер запустится на http://localhost:3001

## Frontend

1. Перейдите в папку client:

```bash
cd client
```

2. Убедитесь, что установлены зависимости:

```bash
npm install
```

3. Создайте файл `.env` (если нужно):

```
VITE_API_URL=http://localhost:3001
```

4. Запустите клиент:

```bash
npm run dev
```

Клиент запустится на http://localhost:5173

## Функционал

- **Регистрация**: Имя, Email, Пароль
- **Авторизация**: Email, Пароль
- **Профиль**: Редактирование всех данных пользователя (фамилия, телефон, дата рождения, язык, смена пароля)
- **JWT токены**: Автоматически добавляются к защищенным запросам
- При клике на иконку пользователя:
  - Если НЕ авторизован → открывается форма входа
  - Если авторизован → переход на страницу профиля
